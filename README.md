About Vaani Setu

Vaani Setu is a platform designed to act as a bridge for communication, breaking communication barriers with AI. It empowers seamless communication between hearing and hearing-impaired communities through innovative technology. Built by a passionate team of Indian developers, designers, and accessibility advocates, Vaani Setu is committed to making communication barrier-free for India's deaf and hard-of-hearing community. It specializes in Indian Sign Language (ISL) interpretation, helping millions access education, healthcare, employment, and public services with dignity and independence.

Technologies Used:

Framework: React
Build Tool: Vite
Styling: Tailwind CSS
UI Components: Shadcn UI
Language: TypeScript

Pages:

The application consists of the following pages:

Landing Page: The main entry point of the application.
About Page: Provides information about the Vaani Setu project.
Pricing Page: Displays the pricing plans for the service.
Login/Signup Page: Handles user authentication.
Dashboard: The main user dashboard after logging in.
Interpreter Page: The core feature of the application, providing real-time sign language interpretation.
Community Page: A space for users to connect and interact.
Tutorials Page: Offers tutorials and guides on how to use the platform.
Settings Page: Allows users to configure their application settings.
Profile Page: Displays and manages user profile information.

How to Run:

1. Install dependencies:
npm install
2. Run the development server:
npm run dev

Build:

To create a production build, run:

npm run build

The output will be in the `build` directory.
